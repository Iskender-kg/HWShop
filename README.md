# HWShop
**HardWareShop**(сокращенно HWShop)-это веб-приложение, которое представляет из себя онлайн магазин компьютерных комплектующих, предназначенный ознакомления для с ассортиментом магазина. В приложении реализованы потребительскиая и административная области.
___

## Потребительскиая область.
В потребительской области пользователям доступен каталог товаров, где они могут просматривать товары по категориям, а также узнать о наличии товара в магазине.
